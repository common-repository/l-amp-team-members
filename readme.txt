A plugin to display an overview of the members of your team.

== Description ==

A plugin to display an overview of the members of your team, you can sort them by category or display as one big team.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the folder `l-amp-team-members` and it's contents to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[l-amp-tm]` on a page where you want the overview
